# Junjie Tang — Personal Website

Personal website and blog built with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme, hosted on GitHub Pages.

**Live at:** [https://JunjieTang-D1.github.io](https://JunjieTang-D1.github.io)

## About

Sr. Principal at AWS — writing about cloud infrastructure, autonomous driving, machine learning, and generative AI.

## Local Development

```bash
# Install Ruby 3.3+ via rbenv
rbenv install 3.3.6
rbenv global 3.3.6

# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve --port 4000
```

Then open [http://localhost:4000](http://localhost:4000).
